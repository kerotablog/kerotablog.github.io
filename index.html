選択のアミダ - Fate Ladder -

ルート選択と運で未来が変わる、アミダ式ステージ攻略ゲーム。
どの道を選ぶかで運命が変わる、アミダくじ式のステージ攻略ゲームです。
プレイヤーは毎ステージ、複数の縦ルートから進行ルートを選択します。

Game

ファイルのせいとん - File Sorter -

気づけば画面がファイルだらけ。
タッチして整頓しているだけのはずなのに、だんだん余裕がなくなり、なんとなく危なくなり、最後は盛大に崩壊します。
「ファイルのせいとん」は、増え続けるファイルを相手にどこまで持ちこたえられるかを競う、直感操作のアホゲームです。
画面をタッチするとファイルが増え、増えすぎると暴走度が上がり、限界を超えるとシステム崩壊。カオスな整頓が始まります。
ただ連打するだけでは毎回同じにはなりません。今回は妙に余裕がある、今回はなぜか早く危ない、そんな手触りを目指した軽く遊べるカジュアルゲームです。

Tool

不動産収益シミュレーター

物件情報から不動産投資の収益性をすぐに確認できるシンプルなアプリです。
物件価格、月額家賃、金利、ローン年数、月額経費などを入力するだけで、表面利回り、実質利回り、月次キャッシュフロー、返済後CFをその場で自動計算します。
詳細入力では、自己資金、管理費、修繕積立費、固定資産税、その他経費も反映可能。固定資産税は「自動推定」ボタンで簡単に補助入力できます。

Tool

繰上返済シミュレーター

複数ケースを比較しながら、繰上返済の効果を確認できる住宅ローン向けシミュレーターです。
基本入力に加えて、複数のケースを作成して返済条件を並べて確認できます。ボーナス相当の返済や詳細設定にも対応しており、返済タイミングや金額の違いによる影響を見やすく整理できます。
